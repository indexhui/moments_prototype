## 事件詳述

### 電梯事件

小麥在捷運搭電梯時，大家紛紛走進電梯，當小麥轉過來面對門口時，小麥跟後面的路人剛好面對面，小麥心想這人是怎麼回事...

這時候會有選項 { 忍住，電梯很快就到了 } 或 { 轉過來，背對他 }，選擇後會有不同的對話。

if 選擇 { 忍住，電梯很快就到了 }
小麥心情會受到影響，這人是怎麼回事啊 （大幅降低情緒？）

if 選擇 { 轉過來，背對他 }
後面是牆壁，轉過來後會看到牆壁，心情會稍微好一點（小幅降低情緒？）

### 走路閃人事件

小麥在街頭走路時，突然看到一個人走過來，小麥往左的時候他也往左，小麥往右的時候他也往右，眼見越來越近了...

這時候會有選項 { 直直走！他會閃開的！} 或 { 往旁邊閃！}，選擇後會有不同的對話。

if 選擇 { 直直走！他會閃開的！}
小麥會直直走，不會閃開，這時候會有對話：
小麥：「他會閃開的！」
結果他閃開了，小麥覺得自己很有勇氣 （心情上升）

if 選擇 { 往旁邊閃！}
小麥會往旁邊閃，幸好沒有撞在壹起，但覺得心情有一點受到影響 （小幅降低情緒？）

### 一塊錢事件

小麥在便利商店排隊結帳時，前面的人買了好幾百塊東西，結果都用一塊錢結帳，隊伍逐漸大排長龍...

這時候會有選項 { 繼續排隊 } 或 { 離開 }，選擇後會有不同的對話。

if 選擇 { 繼續排隊 }
終於買到東西，心情不會變動
後續到達公司時會再出現對話，因為排隊等太久遲到，被扣錢

if 選擇 { 離開 }
沒有買到東西，心情會受到影響 （大幅降低情緒？）
